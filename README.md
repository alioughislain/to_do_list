## 🎯 Aperçu

Une application web de **liste de tâches** complète et intuitive, développée en **JavaScript Vanilla** avec une interface utilisateur moderne et responsive.

![Demo](screenshots/demo.gif)

## ✨ Fonctionnalités

### 📝 Gestion des Tâches
- ✅ Ajouter, supprimer, éditer des tâches
- 📊 Compteur dynamique des tâches
- 🎯 Validation des saisies en temps réel

### 💾 Persistance
- 💿 Sauvegarde automatique dans localStorage
- 🔄 Restauration des données au chargement

### 🎨 Interface
- 🌓 Thème sombre/clair
- 📱 Design responsive
- ✨ Animations fluides

### 🔍 Avancées
- 🔎 Recherche en temps réel
- 📂 Catégories personnalisables
- 🚩 Priorités (Haute, Moyenne, Basse)
- 📅 Dates d'échéance
- 🔄 Drag & Drop
- 📥 Export/Import JSON

## 🛠️ Technologies

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript ES6+
- Font Awesome 6.4.0
- localStorage API
